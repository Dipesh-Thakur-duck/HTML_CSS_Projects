🎯 Bootstrap Grid Padding & Background Color Mystery

Understanding why your background extends to the page edge when applied to grid columns

❌ The Problem: Background on Grid Column

When you apply a background color directly to the grid column element (e.g., <div class="col-12 col-md-6">), the background extends unexpectedly to the left edge of the page.

❌ Applying to Column

<div class="col-12 col-md-6 uni-syllabus-container"> <p>Content here</p> </div>
⚠️ Background stretches to the left edge of the page

✅ Applying to Child Div

<div class="col-12 col-md-6"> <div class="uni-syllabus-container"> <p>Content here</p> </div> </div>
✓ Background stays nicely inside the card
🔍 Why This Happens: The Bootstrap Grid System
Bootstrap uses a gutter system for spacing:
┌─────────────────────────────────────────────────┐ │ ROW (negative margin) │ │ ┌────────────┐ ┌────────────┐ ┌────────────┐ │ │ │ COLUMN │ │ COLUMN │ │ COLUMN │ │ │ │ (padding) │ │ (padding) │ │ (padding) │ │ │ │ ┌────────┐ │ │ ┌────────┐ │ │ ┌────────┐ │ │ │ │ │CONTENT │ │ │ │CONTENT │ │ │ │CONTENT │ │ │ │ │ └────────┘ │ │ └────────┘ │ │ └────────┘ │ │ │ └────────────┘ └────────────┘ └────────────┘ │ └─────────────────────────────────────────────────┘

Key Points:

  • The <div class="row g-3"> applies negative margins
  • Each <div class="col-*"> has padding (gutter value / 2 on each side)
  • The negative margin on the row offsets the column padding
🔑 The Issue: When you apply a background color to the column, it fills the entire column width. The row's negative margins create a visual effect where the background appears to extend beyond the container.

📝 Technical Breakdown
When you use g-3 (gutter = 12px on each side):
/* Row applies negative margin */ .row.g-3 { margin-right: -6px; /* Half of gutter */ margin-left: -6px; /* Half of gutter */ } /* Each column has positive padding */ .col-* { padding-right: 6px; /* Half of gutter */ padding-left: 6px; /* Half of gutter */ } /* Problem: Background on column */ .col-12.uni-syllabus-container { background-color: white; padding: 20px; /* Background fills entire column width including the original padding area */ }
/* Solution: Background on child */ .col-12 { padding-right: 6px; padding-left: 6px; } .col-12 > .uni-syllabus-container { background-color: white; padding: 20px; /* Background is constrained to child div Row's negative margins don't affect it */ }
✅ Best Practice for Bootstrap Cards

Always apply background colors and card styling to child elements, not directly to grid columns.

This keeps your cards properly spaced within the Bootstrap grid system and prevents unexpected visual extensions.

💡 Applied to Your Code
Current Working Structure (✅ CORRECT):
<div class="row g-3"> <div class="col-12 col-md-6 col-lg-4"> <div class="uni-syllabus-container p-4"> <!-- Background here ✓ --> <div class="d-flex justify-content-between"> <span class="uni-year">YEAR I</span> <span class="uni-credit">15 Cr.</span> </div> <p class="uni-semester">Semester I</p> <!-- More content --> </div> </div> </div>

✅ This is the correct approach! Your cards will have proper spacing and won't extend unexpectedly.
